项目分类
    一、SCADA系统(GeneralFrameSystem):
        功能分类:
            1.登录(账户名,密码,本地/在线)
            2.运行界面(左上方显示实时的测试项目值、左下方显示生产结果记录、右边显示生产信息[时间、型号、工单、条码、操作提示、产品结果、统计信息])
            3.生产日志(MES交互、生产步骤、看板交互三部分)
            4.历史数据(查询条件、分页)
            5.系统设置(PLC参数、PLC点位表(测试项目、条码、生产数据)、MES参数、电子看板、生产配方、打印参数)
            6.用户管理(管理系统登录的用户及分配权限)
            7.数据统计(图表展示)
        开发:
            Winform
            MySQL(系统设置表,用户信息表,测试项目点位表,工单信息表,配方表,MES设置表,PLC参数表,屏蔽设置表
                    历史数据表:数据量大并且只涉及添加查询适合好几个表存储,若涉及删改则放一个表存储合适)
    
    二、电子看板服务器(GeneralFrameBoard):
        功能:接收设备数据并保存到数据库用于电子看板
        开发:
            Winform、MySQL

    三、电子看板/数据大屏(GeneralBoardServer):
        功能:展示数据
        开发:
            C/S:WPF(选用)
            B/S:WebAPI + jQuery / ASP.Net Core
            MySQL(与电子看板服务器共用)

    四、MES系统(GeneralFrameMES):
        功能:提供API访问接口(用户登录验证、条码验证、生产数据上传)
        开发:
            ASP.NET Core WebAPI、MySQL
